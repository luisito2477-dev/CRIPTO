# Cryptography Labs 🔐💻

Este repositorio contiene el conjunto de prácticas desarrolladas para la asignatura de **Criptografía**, enfocadas en la implementación de algoritmos clásicos, asimétricos y herramientas matemáticas modulares utilizando **Java**.

---

## 📋 Resumen del Repositorio

El objetivo de este proyecto es explorar las bases de la seguridad informática, desde los métodos de sustitución más antiguos hasta los estándares modernos como RSA, manteniendo un enfoque en la arquitectura de software limpia (MVC) y el rendimiento matemático.

---

## 🛠️ Stack Tecnológico General

*   **Lenguaje:** Java 17+
*   **Interfaz Gráfica:** Java Swing / NetBeans GUI Builder.
*   **Gestión de Archivos:** Java NIO (Files, Paths).
*   **Seguridad:** Java Cryptography Architecture (JCA).
*   **Herramientas Matemáticas:** Aritmética modular personalizada y algoritmos de Euclides.
